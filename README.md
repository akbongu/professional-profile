## AI Tools Used

- **Claude (Anthropic)** — architecture, code generation, SKILL.md authoring
- **Claude Code** — agentic CLI for full-repo edits and scaffolding
- **Cursor** — AI-native editor for TypeScript & Python
- **GitHub Copilot** — in-editor suggestions and test scaffolding
- **AWS Bedrock** — cloud-native LLM hosting
- **OpenAI GPT-4o-mini** — core LLM in the RAG pipeline

## Agent Instruction Files

Each project ships with structured AI agent files:
- `SKILL.md` — operational skills, architecture maps, credential rules, tagging specs
- `AGENTS.md` — per-repo agent rules (e.g. "read Next.js docs before writing code")
- `technology-resiliency.SKILL.md` — DR resiliency agent definition with expected outputs
